# Phase 31.5: Model Selection Lifecycle

Phase 31.5 modernizes the release-pinned Phase 30 model mapping before Guided
Repository Update begins. It gives primary workflow stages and delegated roles
one deterministic, reviewable model policy with exact client resolutions,
explicit capability status, consented availability checks, and safe upgrade
provenance.
